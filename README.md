## 🔧 Usage

```sh
git clone https://github.com/sudoga/self-hosting/
cd self-hosting
# Downloading the static files for the gitea instance
make get_static 
docker-compose up --build -d
```

## ⛕ Routes

**calibre:** `localhost/calibre`

**gitea:** `localhost/gitea`
